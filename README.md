# Scramb Jirion

## How do I install these formulae?

`brew install scramb/jirion/<formula>`

Or `brew tap scramb/jirion` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "scramb/jirion"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
